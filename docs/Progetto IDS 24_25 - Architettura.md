# Architettura

## Livelli
- Controller: gestisce richieste HTTP (REST)
- Service: incapsula la logica applicativa
- Repository: interagisce con il database
- Model: rappresenta il dominio

## Pattern architetturali
- MVC
- Repository Pattern
- DTO (per separare dominio da interfaccia)

## Tecnologie
- Java 21
- Spring Boot
- Spring Data JPA (Hibernate)
- REST API
- Gradle
